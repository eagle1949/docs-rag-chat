# HuggingFace 文本嵌入（embeddings）学习总结

## 1. HuggingFace 远程嵌入

### 是什么

通过 HuggingFace API 调用云端 embedding 模型，将文本转换为向量，无需本地下载模型文件。

### 有什么用

- 快速验证模型效果
- 无需本地算力资源
- 适合开发测试环境

### 示例

```python
from langchain_huggingface import HuggingFaceEndpointEmbeddings
import dotenv

dotenv.load_dotenv()

embeddings = HuggingFaceEndpointEmbeddings(
    model="sentence-transformers/all-MiniLM-L12-v2"
)

vector = embeddings.embed_query("你好，我是穆小课，我喜欢打篮球游泳")
# 输出：[0.123, -0.456, ...]  384维向量
```

---

## 2. HuggingFace 本地嵌入

### 是什么

将 HuggingFace 模型下载到本地运行，通过 `HuggingFaceEmbeddings` 完成离线文本向量化。

### 有什么用

- 数据隐私安全（文本不上传）
- 无网络依赖
- 可批量处理降成本
- 支持自定义缓存路径

### 示例

```python
from langchain_huggingface import HuggingFaceEmbeddings

embeddings = HuggingFaceEmbeddings(
    model_name="sentence-transformers/all-MiniLM-L12-v2",
    cache_folder="./embeddings/"  # 模型缓存目录
)

vector = embeddings.embed_query("你好，我是慕小课，我喜欢打篮球游泳")
# 输出：[0.123, -0.456, ...]  384维向量
```

---

## 3. 百度千帆文本嵌入

### 是什么

调用百度千帆平台的 embedding API，将中文文本转换为向量表示。

### 有什么用

- 中文场景优化
- 国产化 API 替代方案
- 配额管理灵活

### 示例

```python
from langchain_community.embeddings.baidu_qianfan_endpoint import QianfanEmbeddingsEndpoint
import dotenv

dotenv.load_dotenv()

embeddings = QianfanEmbeddingsEndpoint()

vector = embeddings.embed_query("我叫慕小课，我喜欢打篮球游泳")
# 输出：百度返回的向量表示
```

---

## 总结对比

| 方式             | 优势               | 适用场景     |
| ---------------- | ------------------ | ------------ |
| HuggingFace 远程 | 零配置，即开即用   | 快速验证原型 |
| HuggingFace 本地 | 隐私安全，离线可用 | 生产环境部署 |
| 百度千帆         | 中文优化，合规友好 | 国内业务场景 |
