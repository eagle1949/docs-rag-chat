# 今日功能更新（2026-03-11）

本文汇总今天围绕本地文档 RAG 系统完成的功能与改动点，便于回顾与对外同步。

## 1. RAG 后端能力增强

- 支持按 `app_id` 隔离知识库数据：
  - 文档目录、FAISS 索引、文档元数据、会话记忆均按应用隔离。
- 文档管理增强：
  - 文档列表查询（含 `document_id / filename / size / chunks / created_at`）。
  - 删除指定文档（同时删除向量索引和本地存储文件）。
- 上传能力增强：
  - 文件上传新增 `pdf` 支持（原有 `md / txt` 继续支持）。
  - 新增 URL 解析入库能力（抓取页面后切分并向量化）。
- 多轮会话能力增强：
  - 基于 `session_id` 的会话记忆。
  - 支持清空指定会话记忆。
  - 引入摘要缓存记忆（ConversationSummaryBufferMemory）。
- 问答策略优化：
  - 对“我叫什么”等会话记忆类问题优先走历史记忆。
  - 文档检索失败时可结合历史对话继续尝试回答。

## 2. SSE 流式问答

- 新增流式接口：
  - `POST /apps/<app_id>/rag/ask/stream`
  - 兼容默认路由 `POST /rag/ask/stream`
- SSE 事件设计：
  - `event: token`：增量输出文本片段。
  - `event: done`：输出结束并返回 `session_id + sources`。
  - `event: error`：流式异常信息。
- 响应头完善：
  - `text/event-stream`
  - `Cache-Control: no-cache`
  - `Connection: keep-alive`
  - `X-Accel-Buffering: no`

## 3. 前端 RAG 页面升级

- RAG 聊天改为流式渲染：
  - 接收多少 token，页面就追加渲染多少内容。
  - 修复 token 前导空格被吞导致英文单词粘连问题。
- 对话显示增强：
  - 回答区支持 Markdown 渲染（与 DetailView 体验对齐）。
  - 流式输出期间显示打字光标动画。
  - 对话区自动滚动到底部。
- 页面交互增强：
  - 上传区支持 `pdf` 文件选择。
  - 新增 URL 输入并触发解析入库。
- 布局优化：
  - 三栏布局改为更稳健的响应式网格。
  - 中间聊天区、左右侧栏支持更合理的滚动与收缩。

## 4. 工程与运行相关

- 新增/更新运行方式：
  - 提供 `Backend/run_backend.bat` 便于一键启动后端。
- Git 忽略规则完善：
  - 忽略 `__pycache__` 与 `.pyc` 缓存文件。

## 5. 当前状态说明

- 关键功能（多知识库、文档上传/删除、URL 解析、SSE 流式问答、多轮记忆）已可用。
